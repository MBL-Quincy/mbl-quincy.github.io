耗尽一整天配置 VS Code 2018-02-02

GitHub 的 Atom 看起来后劲非常不足，最近有很多包都陷入了缺少维护和更新的状态。我最近需要使用 Java 开发项目，而 Atom 对 Java 的自动补全或者项目管理都无法正常运转。但 Jetbrains 系列的 IDE 又太复杂，于是我用一整天的时间切换到了 Visual Studio Code，并设置到了一个比较顺手的状态。

安装

使用 macOS 系统安装 VS Code 可以通过官网下载安装包。

实用插件

  1. Atom Keymap: 微软官方针对 Atom 迁移到 VS Code 的用户推出的快捷键映射工具，提供了一些实用快捷键，比如 Command-\ 关闭开启边栏，Shift-Ctrl-M 预览 Markdown 等。
  2. Markdown Preview Enhanced：增强的 Markdown 预览工具，支持 LaTeX 渲染。
