蝉翼

发表于 2015-08-11 | 分类于 琐记

我所说的蝉翼，真切的，就是蝉的一只翅膀。今天上午我在公园的石头路上看到了这只翅膀，旁边什么都没有，也看不到作为主人的蝉的尸身。这条石头路的两侧的树还是绿色的，响着一叠又一叠的蝉鸣，他们谁也不知道自己的一个同伴坠落了树枝，然后暴毙在灼人的阳光下。他们鸣叫着，就像哀号着，但他们谁也不知道这件事。

我害怕蝉这样规模大小的昆虫。在高考的两天前我见过类似大小的虫子，就是我在学校旁边租来的老房子里。夜里我电话里放着郭德纲的相声，我准备入睡，但我看到一个黑影在床边挪来挪去。我惊地后退，不敢开灯看它的样子，也不敢动弹。我叫喊着外婆，外婆过来开了灯，拿着一张卫生纸把它干掉了。

这样的事情发生了不止一次，我在我的床边会面了各种大小的黑色的虫，可直到最后一次还是没能敢于杀死它。亏得高考即刻来到了，我即刻就离开了这件屋子。否则我想哪一天，我也要上演一场变形记。

所以今天这枚蝉翼，我也没能敢捡它起来，看看人们常说的“薄如蝉翼”是怎样的薄。

白先勇短篇小说《青春》

发表于 2015-08-04 | 分类于 读书

太阳已经升到正中了，老画家还没有在画布上涂下他的第一笔，日光像烧得白热的熔浆，一块块甩下来，粘在海面及沙滩上，海水泛着亮白的热光，沙粒也闪着亮白的热光：沙滩上的大岩石不停的在冒水烟，烟色热得发蓝。整个海湾都快被蒸化了。

老画家紧捏住画笔，全神贯注的想将颜料涂到画布上去，可是每当笔接近布面时，一阵痉挛抖得他整个手臂都控制不住了。额头上的汗水又开始一滴一滴落到了他的调色盘上，阳光劈头劈脸的刷下来 ，四处反射着强烈的光芒，他感到了一阵白色的昏眩。

站在岩石的少年模特儿已经褪去衣服，赤裸着身子摆出了一个他所需要的姿势，在等着他涂下他的第一笔，然而他的手却不停的在空中战栗着。

早上醒来的时候，阳光从窗外照在他的身上。一睁开眼睛，他就觉得心里有一阵罕有的欲望在激荡着，像阳光一般，热烘烘的往外迸挤，他想画，想抓，想去捉捕一些已经失去几十年了的东西，他跳起来，气喘喘的奔到镜前，将头上变白了的头发撮住，一根根连皮带肉拔掉，把雪花膏厚厚的糊到脸上，一层又一层，直到脸上的皱纹全部遮去为止，然后将一件学生时代红黑花格的绸衬衫及一条白短裤，紧绷绷的箍到身上去。镜中现出了一个面色惨白，小腹箍得分开上下两段的怪人，可是他不管自己丑怪的模样，他要变得年轻，至少在这一天；他已经等了许多年了，自从第一根白发在他头上出现起，他就盼望着这阵想画想抓的欲望。他一定要在这天完成他最后的杰作，那将是他生命的延长，他的白发及皱纹的补偿。

雷雨

发表于 2015-08-02 | 分类于 琐记

钟立风说，郁冬给自己的歌起名字常会拿来经典电影、小说的名字来用。看到这话的时候，我表弟在旁边说这不是抄袭吗？我却心里泛起一点温暖。大概这样会我们更加辽远的想象，将熟知的事物延伸吧。

似乎每一件文艺作品，我们感受的都不是它们本身，而是特定时空下的自己。就像我现在回忆塞万提斯的《堂吉诃德》的时候就会想起某个冬天在北京的郊外度过的日子；回忆张艺谋的《山楂树之恋》却想起和表弟嬉戏打闹的傍晚；回忆刘若英的《听说》又想到从南沙河南头走到北头再走到学校的每个早上。南沙河的桥修了好久，跨越了我的整个高三，装运水泥的车每天早晨和我一起走过这条路，发出的巨大轰鸣声让我不得不调高耳机的音量才能掩盖的住。我不骑自行车，却也看到辐条若隐若现的转动，听到轮胎与车身的摩擦和止不住的车铃响。

昨天雷声好大，我房间的小窗子外面闪着灯光，却不时被骤亮的雷电给掩盖下去。我有点害怕，又知道拉上窗帘也是无济于事的，苍天的这些怒气是无论如何要撒给我们看到的。我把身子扭到面对墙的一边，白色的墙又毫无保留地把雷电的光反映在整个屋子里。

游园惊梦

发表于 2015-07-29 | 分类于 琐记

当我讲起游园惊梦这个名字，我是在说汤显祖的一出戏，白先勇的一则短篇小说，还是郭富城的一首情歌呢？这个名字好像本身已经具有了梦的意味，让这些所有的符号交织出现，错综融合。所以当我讲起这个名字，我心中就多了一团撕扯不开的麻线团子，然后忍不住要往窗外扔掉它，再也不叫我烦心。就像，我们永远搞不清楚多年前让小虎队名声大噪的是一颗苹果，还是一次新年。

夜晚的公园小路上，现在越来越多的是自然而又诡异的气氛。吃过晚饭，嫌弃肥胖的女人们、男人们会从家里走出来，只穿着着睡衣在公园里快走。我家的西边附近是一个山上的公园，走到坡度陡的地方总能看见几个赤裸着上身的男人大步向上跋涉着，腰带似乎要把圆滚滚的肚子挤压成两个圆滚滚了。有好多人就在这里一圈一圈的走着，彼此不讲话，只是走，就像无声的海浪，永远不停地翻滚着，谁也不知道究竟在翻滚什么。尤其是在夜里，是在灯光昏暗的山中小径，人们漫无目的的往前走，安静得像沉睡的老虎；走在北侧的一条路上，也只能能听到鞭挞陀螺的巨响，不知是哪位身体强健的老者，又捡起了孩童时的陀螺，拿着鞭子抽挞着。

翻滚

发表于 2015-07-26 | 分类于 琐记

又是一个热得作响的夜晚，这个狭小的房间看着整洁，却无时不在发出杂乱的、扰人入睡的巨声。窗外和窗内都一片黑漆漆的，我只听见了嘈杂的声音，无从知道这些噪声是从何而来。我从床上坐起，把空调按开，感到前胸滴着热汗的皮肤感受着温润的冷气。

我赤裸着身子，一会儿就凉了下来。空调创造了另一种嘈杂的声音，而让原来那中不能琢磨的噪声赶走了。我爱空调的嘈杂，能让我在这样的夏天里清凉下来的我都爱。我甚至都有点觉得发冷了，拿脚勾起了被单，撒网似的裹覆在了身上。

我想着这时我的样子是有些狼狈的。俯身趴在床铺上的身子盖了一层印着粉色花朵的被单，足把我的瘦弱身躯勾勒地更加寒酸。我让被单覆住了脑袋，却在侧向左边的眼睛那里开了个小口子，正好看见一只我的嶙峋的左手，指头上的汗毛在空调风的调教下瑟瑟发抖。这个小窗口外的微光让我看清这只手，也看清它在仿佛弹着琴键一样地在床铺上轻舞，给只有这一点天日可见的我的眼睛看。房间的嘈杂，是这只指头的舞动出的乐符吗？

充满了冷气的房里，我还是不很安分。左边的床铺被我捂热了我就翻滚到右边，让冷气去降温左边的热。右边的床铺也这样被我对待着。谁也不会知道，我在这样炎热的夏天，在我狭小的房间里，狼狈着样子，做这样的翻滚游戏。

狂热

发表于 2015-07-22 | 分类于 琐记

我设想自己去接录取通知书的状态一定会是无比平静的，因为经过三四次确定，对一件事情已经有些厌烦了。可在这个刮着强烈东南风的中午，我还是跳着脚地下了楼，宽大的睡衣被风扬得不听使唤，把这具欢腾的瘦削身体的轮廓刻画出来。

是什么造就了这样的狂热呢？我旋走狂奔在楼梯上的时候，手里抓着只电话，趿拉着塑料拖鞋，十足有着在楼梯上摔倒的危险。我感到就像有个人揪着我的前襟，揪出了深刻的折痕，然后拖着我往前移去，而我何其享受，何其沉浸。同时，我的耳朵里好似有刚刚打碎一片玻璃一样的脆响，伴随着鞋子和水泥楼梯摩擦的声音，凑成了耳边的一套交响。在印象里，这种交响是在国家大剧院里听文工团合唱的时候才能听到的，零落中却有摧人的气派，让人往前赶去。

中午的风很大，因为一场雨在西边的一朵深色的云里积蓄着，没准刚刚在省城那边发了一顿怒气了。风把旁边矮楼的玻璃打得哆哆嗦嗦，也把我的眼睛吹打地眯成一条线，嘴也吹得歪了。在快递单上签字的时候我直想用左手把嘴捂住来缓和这种扭曲，可左手正扶着我的宝贝，颜面还是不顾了。

这样的拆封时刻好像是很神圣的，所以我该想到去厕所把手洗洗干净。玻璃和拖鞋的交响结束了，家中安安静静，我才发觉电话里的音乐电台一直没有关。电话里正放着张震岳的《爱的初体验》，我撕扯开红色的快递单子，准备津津有味地读一会儿我的未来。

楼

发表于 2015-07-17 | 分类于 琐记

我的窗口前是一片矮楼，一齐都长着光秃秃的灰色屋顶。雨后这些屋顶闪着暗哑光线的雨迹青一块紫一块，像是雨用力打出的伤口。两排矮楼之间露出的狭窄公路，好像是这片灰色还活着的唯一证明。

尤其到了晚上，灰色变成了黑色，楼与楼之间的空子也看不分明，我只看见这条公路，湿气未干的水泥地上模糊地倒映着夹缝中穿梭的汽车们，来来往往，拥挤着，构成这个城市没有人注意的一条甬道，流淌着看似无关紧要的血液。

照理说，落地窗外应该是成片的玻璃幕墙，抬头就能望到触及云端的尖塔，比如上海，比如纽约。Danny Boy说，在中国与人喝酒总是勾肩搭背，来到纽约这样的都市就是要享受渺小与孤独；面对成群的摩天楼宇，起初会恐惧，进而会想壮阔地拥抱。可我的窗前是没有高楼的，一片矮楼就像起伏的大陆，闪着灯光的公路就像峡谷。我站在落地窗前，这个没有一座摩天楼的城市，一眼看去就到边了，北边筹谋着盖一座高楼的塔吊就是这个夜晚的消失点。

哦，我说谎了，这里是有高楼的。我踩的这一座，不就是吗。

红豆

发表于 2015-07-16 | 分类于 琐记

母亲又熬了整整一锅的豆汤，她说绿豆太凉，得搁些红豆来调和。刚刚打开的锅盆里，有几粒爆了壳的绿豆漂浮在水面，红豆则没了踪影。母亲要我把它当水喝，要快快把这一盆喝光。她说夏天什么都败坏得快，放在外头，一天就酸掉了。我说太烫了，一会儿再喝。

今天下雨了，上午一直阴着。昨天中午睡觉的时候，窗外面的雷声震天响，我的心被它吓得震动，脚把被子踹开，却也闷得枕头湿了大半。可是说来奇怪，响雷的时候，我和雷一齐震动，而除了我和雷之外的东西，比如桌椅，比如柜子，它们好像装作什么事都没有发生，安安静静地在那里站着。昨天的雷没有带来一场雨，今天早上的雨悄声无息地来。

我对母亲说，这豆汤大概是不会坏的，尤其是搁进去了红豆，尤其是在这样的下雨天。我舀起了一瓢，绿豆们争相从瓢子边缘滑落下去，红豆还是在瓢底从不动弹。我使劲摇那个瓢子，暗红色的汤底坠落，热气从四周翻涌上升。在雨天，我有点忘记了这是夏天。所以被那一团热气温暖着，我没有一点厌恶。我却有点厌恶那颗红豆了，像一只老乌龟那样不舍得动弹一下。我恨不得它赶快败坏掉，长出白色的臭毛。

可是啊，有一种红豆，不论是什么天气或者季节，它是永远不会坏的。

夜想

发表于 2015-07-14 | 分类于 琐记

晚饭过后，我翻到了《皮囊》来看，电视剧的吵闹胜过了隔壁装修的叮叮咣咣。沙发的左边是灯火通明的落地窗，却只有一个小窗口装了纱帘，可以打开通风。沙发很长，能够让我四仰八叉地横卧在上头。可无论怎样伸展自己的身体，刚刚吃进去的饭团就像下水道塞子一样堵在胃口那里，好像再喝一点水就会阻塞在喉咙上。从前胖的时候，似乎从来都没有这样的感受。我放弃了挣扎，我知道饭团过一会儿就会瘪下去，又恢复了通畅。

电视突然安静了，原来是Emily在向Naomi恳求着什么，池塘里只有滴答的水声。隔壁工人的叫喊和锤子砸墙的巨响有变得突出起来。我有点生气，把电视剧暂停掉，然后回去到卧室。九点了，已经有点犯困了。

九点就犯困了，这样的人现在很少了吧。高三的时候，老师和母亲总怨我每天太早睡下，总是放了晚课回到家就躺在床上了。我无力辩驳，年轻的心只教我“自行其是”。并且，关于“努力”的辩论，在每一个周末母亲来探望的时候总会开启一番。我想这些无意义的争吵都已经结束了，所谓努力以及自强不息的理论在我这里都告破。

恶齿

家乡这里的人，在我看来长相都有些粗野，尤以倔强突出的双唇为特色。我也不例外，小时候我总是合不拢自己的嘴，牙齿外翻地有些太放肆了，上中学的时候被妈妈押着看了三年的牙医才稍显好转。有一次，医生把固定在我牙齿上的铁块子撬走，塞进去一只可以卸下来的塑料壳，说是要定型。我也不知道那个壳子很贵，走在街道上，就把它卸下来，扔进垃圾箱里，过后再没有去医牙。要我说，那个牙医的牙齿很难看，怎么不先把他自己医好，反而先来医我呢。

铁路和胖男人

发表于 2015-06-27 | 分类于 琐记

这个城市有时候还是能看得入眼去的，比如车窗外的一半货运列车。它很脏，绿色的篷布下蒙着炭块，奔驰在山腰的铁路上。我看不到列车的头和尾，好像它要把这座城市刺穿，留下蒸汽时代的一道流烟。灰白色的漆、暗红色的车标，它就这样与我背道而驰了，我终于看到了它的尾，硕大的铁钩给它殿后，似乎充满着不甘心。

另一边，就是干枯的河流了，它也许不能叫做河流，只是恶草丛生的沟壑。闹市区里的那段，有一人多高的铁管子往沟里注水，西边的那段因为少有人来看风景，野草还是很放肆。可是这边，偏有一个蹬着皮鞋的胖男人，一只脚踹在河堤上，大口啃着红瓤西瓜。他把T恤捋起来，肚皮就弹了出来，把衣衫正好卡在胸口那里。远远看，那个肚皮就像另一只大西瓜那么厚实，再用力一点就要把裤子搞破了。

野草不怀好意地摇摆，嘲笑一只肚皮的圆滚。胖男人背对着夕阳，背对着他放倒在地上的摩托车，背对着蒸汽缠绕的铁路，也背对着灰蒙蒙的矮山。他只是啃着自己的西瓜，不时拿起暗绿色衣服的一角，擦拭沾满了红色的果汁的嘴角。

酒

发表于 2015-06-23 | 分类于 琐记

我的家住在九层，出门要做电梯，回家也要坐电梯。电梯里有时候会有一个袒露着肚皮的男人，稀疏的头发摇摇欲坠地耷拉在肥脑袋上，嘴里吐着酒精味儿的哀叹。我想他是有烦恼的，或许是那种听起来俗不可耐的烦恼，但只要令一个人很失落，烦恼也就算是很重要了吧。麦子和酒花在酒缸里沉浸了那么多时日，壳子泡得好软，似乎再久一点，就化了。酒的温情大概就是来自于这种柔软吧，抚慰着失落的人。
