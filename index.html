GitHub 的 Atom 看起来后劲非常不足，最近有很多包都陷入了缺少维护和更新的状态。我最近需要使用 Java 开发项目，而 Atom 对 Java 的自动补全或者项目管理都无法正常运转。但 Jetbrains 系列的 IDE 又太复杂，于是我用一整天的时间切换到了 Visual Studio Code，并设置到了一个比较顺手的状态。

在长江旁边发现自己

发表于 2016-09-26 | 分类于 琐记

其实一年后的今天，我还是搞不清很多东西。我不知道该如何相处朋友，不知道怎样思考学业，也不知道自己有什么值得全力以赴的愿望。

九月二十二号的傍晚，我乘着嘎吱作响的二十六寸自行车，沿着中原路向北走。走到尽头是军工路，那里大致是长江入海口的边缘，从一道小径刺穿进来，行道树和暖色调路灯辉映着，川流不息的卡车叫喊出巨大的嘶吼。宽阔路面的南侧是整齐码放如山高的集装箱仓库群，远远看去就像睡在平原尽头的锈迹斑斑的巨大胶囊；北侧布满了港口的巨大吊台，可能是这片陆地伸向海洋的骄傲触手。这里是一片港口，在我去年的秋天第一次骑行到这里的时候，我就被这片充满了机械感的景象给迷住了。

第一次来到这里时，看到地图上近在咫尺的长江水，却怎么也看不到。港口上林立的货运公司把我和长江隔了一道灰色的裂缝，让油轮击打江水的声音回彻云端，但不让人看到海洋的面目。我没有见到长江，就沿着繁闹的淞沪路回去了。入学以来我许下不少愿望，这最后一个也毫无悬念地破灭了。

我的第一年其实就是这样过来的。学期深入下去的时候，琐碎的事情好多，学业上做出很多尝试和努力却总是不明就里、一知半解。于是，琐事繁重的时候骑车向南游荡，习题艰难的时候向北穿行；扔下烦心的所有事情去，全然不管这样任性的后果和更大的痛苦。好多个傍晚，我独自将这里的每一座路灯都熟悉起来：我偷偷盗走了新江湾城的某个高档小区的垃圾收集处的大门钥匙、在淞沪路上的一棵行道树上用圆珠笔画了一个小人、在中原路麦当劳那里索要了很多垫餐纸，也不知道算不算是挖社会主义墙角的无耻行为。在游荡骑行的路上，我不停地思考着大学生活该是怎样，明天又该如何挨下去，可总也没有答案。需要思考的东西变得那么多，我变得很无措。

教师节有感：少一分任性，多一分爱

发表于 2016-09-10 | 分类于 习作

我的眼前如同忽然倾下一盆油汤似的，腻腻的，怎么也看不清楚。耳边也似奔过上千个怒吼着的身着铠甲的士兵，炮声连连震天。当我明白过来我胳膊上的一条长长的红印子是长久伏在粉色封皮教科书上压出的轻轻瘀血，我就用力揉了揉眼睛。铃声在响，身边的同学都拾掇书包离开教室。士兵的咆哮声不见了，取而代之的是活动椅弹起的时候强烈的碰撞声。

我又睡了一节课，在一百个人教室的倒数第三排，靠窗这个最不起眼的位子上。这课我怎么也听不懂，老师讲过四十分钟，我才能刚刚把第一分钟的内容想得明白。后半程，便把课本一合，任春秋大梦带我去南极洲走了好几遭。教室慢慢空了，只留下几十个人在这里呼吸吞吐留下的燥热气息。老师一个人在讲台上，把粉笔整齐放回盒子里，拿起几张讲义和几本书夹在腋窝那里。他回头看了看黑板上的内容，然后径直走出去。窗帘在我身边还在摇晃，老师腰际的钥匙，敲打作响，构成下午三点钟这个楼道里唯一的声音。

我像这样，在过去的一年“混”过去不少课程。和我同行的还有总坐在第五排看球赛的几个哥们儿，总坐在第七排读漫画的女孩儿，还有每十分钟必须要走一趟厕所的小胖墩。我们听了一个学期的课也不会在路上准确的认出老师的脸，在课上甚至没有与他说过一句话。老师说“你们明白了吗？”我们岿然不动。转笔的，看手机的，上厕所的，睡觉的，都岿然。

有时候我觉得老师可怜了一点，孤独了一点。用力教学复杂的知识，台下真正在听的却总是寥寥。云里雾里如我，便破罐子破摔了，更不想与老师有任何接触。但我总觉得愧疚。如果我用力一点，勇敢一点。坐到前排去，跟老师请教；或者常与老师交谈驳诘，说不定更好一点，能更了解老师，了解他平淡行事背后的心灵波涛和用心良苦。遗憾的是，很多时候，我们从第一次在课堂上睡着之后，跟老师的距离就越来越远了。

竹箫

发表于 2015-12-05 | 分类于 习作

如果有个男孩子，每天夜里在面对着窗子的广场上用竹箫吹奏起同一支音乐，即使你与他不熟识，你是否也能听出他的心事？不说别的，我住的这幢楼房里就有这样一个男孩子。夜里我去广场上晾衣服的时候，从这幢楼的第十七层飘来的悠扬箫声，总提醒我这幢楼的生动。箫的声音不像是西洋管乐的娇艳或是宏伟，它空空的，从左耳进去又从右耳出来；好像说了好多东西，但好像什么也没说。西洋的乐器是蓝色眼睛和深陷下去的眼窝，而箫就像是喝醉了二锅头的胖汉，在酒馆里宿醉，自己叨念着无人可诉的烦忧。

每到广场窗子外头的那两盏路灯亮起，不出十分钟，竹箫的乐音就化作了初冬的风，冷得蜇心，吹打向挂在每一层广场上的湿衣服，吹打向住在每一层楼的盲目身影。这幢楼里装着形形色色的人物，比如我的对面是一个在手机修理铺做工的十一岁男生；楼下的那个小伙子在附近一家师范学校念书，还有十七层那位没有见过面的竹箫。不过，这楼里的人们都是一样：每天清晨若是找不到干净的衣服，就把广场上还浸着的湿褂子摘下来裹在裸体上，踏上两只鞋子出门去。

辍笔半月，其间多次翻开电脑想打些字儿进去，都在苦苦憋出一百多字后全部删掉，作罢念想。固然，生活的日渐规律让我为笔下的这种乏味找一个理由，但坐下来想，还是心中矫揉念头消散，造作的愿望也逐渐淡了下去。这些纠到深处都是读书少的缘故吧。新的一辑《读库》，到现在还显着崭新。

几天前，读库的微博上放出了这周日在上海将有一场十周年纪念活动的消息。我昨晚与同学谈天吃饭到深夜，回到寝室才翻到这则消息，念及三年读《读库》的情谊和对主编张立宪老师（人称老六）的仰慕，我决然准备前往。于是今天辗转地铁来到江苏路上的“东方收藏”书画院，聆听人生导师的教诲。

大学以来，与新同学谈论最多的是各种虚头巴脑的活动，与旧同学谈论最多的是包袱不断的八卦和拉郎配。七八个人凑在一起常喜欢调侃牵着手走在街边的一对同学，一遍又一遍地讲“以前没看出来你俩”的诸多玩笑；总与一位饱受长达三个月的异地恋之苦的同学并肩行走，耳听到的都是委屈和抱怨，还有就是“你为什么不找个女朋友”之类的问题。我不得不承认这些话题的引人入胜，妙趣横生，我只要听到就会竖起耳朵，怎么也听不够。可是就在这个质疑人生、思辨重于泰山的年龄，能够探讨心灵鸡汤的好朋友却鲜见于眼前，可能重要的话都去对男女朋友说了。倒是远在太平洋对岸读书的同学总有灼见与思考拿来交流，不胜感激。

你在我对面（续）

发表于 2015-10-25 | 分类于 习作

注：本文为《你在我对面》的续集

旧物

在家里歇了一个月之后，我把卧房门上的小熊玩具洗洗干净，带到火车上走了。这些天来，我拼了命地爱上了屋子里的一切旧物。十几年的生命很年轻，但我惊异于年轻的日子里积累下的丰富。我甚至在床下发现了四岁时候玩过的荧光卡片，然后找了一只拖布棍子硬是把它挑了出来。再次出门之前，我大概把一切忘掉的事情都想了起来。包括那个荧光卡片，它是我和小学的一个姓李的同学打赌那天会不会下雨，他输了，然后买给我的。

还有电影。上周三的下午我伏在电脑跟前看《泰坦尼克号》，父亲走进来说，小时候就带你去看过啊。可是我一点也不记得，对我来说每个镜头都是新的，从没见过的。可就在父亲讲过之后，这部电影也蒙上了一层旧物的尘土，我渴求扫除干净，抓住那里边的曾经的我。

我曾经以为睹物思旧是一种很没道理的事情，年轻的心一个劲儿地想向外追求，看不到身后的影子。我想我是偶然回头的那个，拼了命地爱上了屋子里的一切旧物，然后把忘掉的事都想了起来。从前我躺在潮热的床上，对于短暂的过去，我只记得有两个玻璃窗。这次出门和以往不同，年轻的心上负载了太多记忆，大概也就飞不快了。

时间，时间。米开朗琪罗好像在什么地方说过，他在幼年就感觉到时间具体的流动，是一种知觉无法触及的知觉。有的时候我的时间很复杂，对我来说，包络了窗户旁边光影的变幻、我和李成的相继离开，还有我带有哭泣的一次又一次的歌声。时间的流逝不是拍一个Time-Lapse就能够表现清楚的，因为同一个桥段，不同的心会对时间会有迥然的变换，而所有的心交织在一起，也许才叫做时间吧。

哲学我是不懂的，也许正因如此才有这样多的困局让我来面对。我回到学校来，这一次，没有小熊的玩具让我抱着我是断定睡不着的。

重舒

发表于 2015-10-25 | 分类于 习作

洋气

专学外语的学生，不论是否愿意承认，眉目动作间总带着点刻意模仿洋人的姿态。读阿拉伯语的，爱在肩上披大块的纱巾；读法语的，把嘴唇涂得厚厚的红红的；专攻英语的女孩子，头发染成棕色甚至金色，眼睑上的黑影画了一圈又一圈，男孩子则特别青睐皮衣和深色的毛料马甲。我刚刚考进清华的时候，觉得身边的这些同学的确是长于打扮和美化自己，长于化自己为异国的身份。不过我想这些姿态不能够称为造作，也不能冠上媚外的帽子。吃一口饭的人，总会被这口饭给感召得变成另外一种样子。

不同的是，和我住在同一间屋子的重舒却始终没有被那些个洋打扮给感化。两年前的那一天我记得清楚，他戴了老式的眼镜，上身披一件马褂，一趟趟把行李运送上楼，然后拿马褂的袖子一挥，揩掉脸上的汗，坐在凳子上长舒一口气。那时的重舒还有一点微胖，可是不论怎样还是盖不下去他眉宇间的气概。我无论在哪里都没有见过这样的眉目，时而英武时而温润，刚刚结识的头几天我甚至不敢直着看他，好像一双眼睛要把一切话都说尽，要把一切的世界给吞进来。

我和一切外语学生一样，不论是否会用得到，入学的时候都恨不得把书架上的英文书全都倒腾到大学宿舍里去。还没发放教材的时候我就已经把柜子旁边的书架塞满了。头一层是三本英文字典，两本西班牙语字典，一本法语字典和一本拉丁文参考书；第二层是一整套的英语写作参考；第三层是自己饶有兴致的书，甚至还把一本薄薄的从没看过的《希伯来语简史》摆在了架上。那时候，我高兴地拥有了自己的一片小天地，晚上睡觉前还要给书桌拍张照才能安眠。重舒那天收拾完行李就一屁股坐在桌子上，那着一本薄薄的小册子翻来翻去，偶尔抬头看看我们的忙乱情状。房间的一角是他的安静，另外三个角落是我们其他三人噼里啪啦把厚厚的书往书架上摆布的巨响。可能在我们看来这种巨响是一种自豪，在他看来书架上空空如也才是一种自豪。
那天晚上睡觉前，我们约定好闹钟的时间，我看到重舒的书架还是一层层裸露的木头，只是把那本小册子扔在了桌子一角。《干校六记》，呵，这哪里是读外语的学生。

跟重舒稍稍熟络了之后，有天我们从饭厅归来，我说，咱们肯定和那些大哥大姐一样，一年以后都变得“洋气”不少。我俩忽然一同学着法语女学生一扭一扭的步态，然后一同大笑起来。

投湖（杭州行有感）

发表于 2015-10-07 | 分类于 琐记

朱光潜说，一个人见到宏伟的东西，如山川，如蓝蓝海洋，首先会产生恐惧，怕被这些巨物给摧毁；但不出一刻，人便被宏伟给卷进去了，内心油然产生极大的归属感，这就是人们敬畏巨物、狂热于巨物的原因。所以，当我再一次踏上堤防，我就抑制不住了要投向汤汤湖水的冲动。夜的西湖就是一口井，井口让水打磨得细密模糊，生生要把人活吞进去，和它一同化为肌理分明的混乱。

混乱，是的，我一向喜欢用混乱来解释身边的一切事情。朱利安•巴恩斯就说过：“我们只追求混沌的享乐，一概不考虑别的选项。”这在我看来是没错的。眼前这摊湖泊就充盈着混乱的气息。还没有到达湖岸的时候，旅行者翘首以盼的是“西湖”这个独一无二的名字，但真正临近了，就不会再有这种念头，竟然连这座湖叫什么名字都忘记了。夜里是没有游船出没的；迷蒙的积雨云下头，湖水泛出咸得发苦的棕黄色，焖出了厚厚的一层水雾，徐徐蒸腾，以求感化天地。我就像俯卧在了一面宽阔细密的皮肤表面：呼吸在暗涌中起伏，毛孔里带着体温的汗雾兀自喷将出来，将攀附其上的一粒尘埃给熔化了进去。

我想叫喊，比见到大海时还要想。但叫喊似乎不适合西湖这样的品性。走了几步之后，我还是激动地喊出来了，尽量放低自己的声音。我对同伴说，还好和你一起来了，若是一个人来，这样的地方没人分享，会憋疯掉的；这里的诸般意味，照相机是捉不进去的。

记一次流动

发表于 2015-10-02 | 分类于 琐记

钟立风好像在他的哪本书里提到过，说生命就该是流动的，他说的流动我想包括了周遭环境和人事的变迁。流动饱含着能量，推动着向前涌去。这一个月过去，我才想起来，我是个大学生了。提醒我明白这一点的是最近无论如何也解不出的数学题目，和费很大劲才能读懂的课本。我有点钦佩高中生，他们真厉害，那么难的题目划拉两笔就解出来了。

不过，刚刚进入新生活的我们这一群人似乎都出了好多问题。两周之前那个名叫“醍醐”的群聊，徐同学把我拽了进去，里边净是我看不太懂的人生思考；昨天午后杨同学在我身边埋怨，宿舍里都是打游戏点外卖的宅男，学了理工科以后，接触的文化熏陶越来越少了；晚上和马同学的一番电话长谈，关于处理同学关系和自我价值的困顿和困倦；还有我怎么也看不懂的书和习题。每个人好像轻轻溪流中的一滴水，一次次碰到了石子，却还得被水流推着向前去。流动的生命是不由着自己的，任何改变也是全然无法预料的。同学对我讲，想暂停下来，好好想一想接下来该怎么过去。我说，就混乱着吧，从没有想的时间。

走进新时代

发表于 2015-09-17 | 分类于 琐记

为了走进一个新的时代，我在自己的角落里做了很多尝试。除了在没有阳台的屋子里晾干几件衣服，还有吃石榴这件大事。那天新识的同学送给我一盘石榴果，红红的堆满了。欣喜之余，我还迷惑。我从没吃过石榴，事实上，水果我很少吃。

盘子的边缘看起来很多水珠，我为了掩盖自己的手足无措，佯装要把那些水珠给抹去。谁想到这些水滴不过是用瓷土堆起来的硬邦邦的假水，灯光下透亮，衬得盘子都新鲜无比。我想我这一步是必须走下去的，我拿了颗石榴，放进嘴里咬下去。室友说，一粒一粒吃，什么时候能吃得完呢。我抓了一把请他示范，他一口含进去，胡乱咀嚼一通，挤出白花花的一团石榴籽。

我自己去尝试的时候，舌头牙齿绞在一处，怎么也不听话。我只能赞叹室友舌头灵巧，而自己仍旧一粒一粒吃掉。我坐在桌子上，透亮明闪的盘子搭在腿上，一粒，一粒，我吃了四十分钟，才把这个时代吃完。
