引言

自高考结束以来，我的个人博客已持续将近半年，文章寥寥，情致如初。数月之内，书籍读的不多，洋洋30余篇日记或是习作倒是记录在册。爱读的两三挚友每每在博客更新后立刻读过，然后与我网络上聊天片刻，我不尽感恩，也快乐于往来驳诘。我的表哥嫌弃我语言华而不实，情感空洞无物，我也承认。年轻的人需要历练，即使今后再打开旧时习作，一定是鄙陋不堪，倒也不是件坏事。

而半年来的收获，除了作一些胡乱篇章，还有一番令人欣然的折腾故事。独立博客不比微博、朋友圈这些方式，台子搭好只等你来唱戏，而是另一番自弹自唱的经历，我凭借粗浅的计算机知识，断续从网络上各方神圣取经，让自己的博客完善起来，稳定下来，有了些收获和进步。

一年近末，博客也该有些定数了，关于技术方面和内容方面的折腾，也该有些准头了。这篇短文我放在首页顶头，作为自己过往之经验，写给自己，写给愿赏光来临的伙伴，也写给打算同我一样开创自己小天地的同仁们。

初衷以及今后之定位

我在高三的时候便有了搭一所自己的博客的念头和计划，缘由自然是心中欲求无从释放，可能青春年龄的少年都有点“浪催”，有的人选择心无旁骛，闷头苦干见天日；也有人想我一样是这种摇摆不定，招摇着求索的小孩。相比于传说中更加困苦的高三学生，我觉得我所处的环境已是颇为轻松，也没法逃出成长中偶有迷惑的窠臼。那时通过习作与语文老师交谈，通过聊天与身边同学往来，每日在一个本儿里记录下一日乾坤，迷惑不减，但兴致更佳盎然。

受高中语文老师的引导，我渐渐明白了读书能够带给人的力量。不知是何缘由，同学老师偶有称赞说我“读书挺多”，但我心内知道，我所读过的书简直太寥寥。右到各方老师与名家推荐的经典文学，左到耳熟能详的通俗小说如《哈利波特》之类，我皆没看过。如此下去，大致会坠入“思而不学则殆”的迷雾中。高中虽然学习理科，但同班的大家都多少明白人文智慧的力量和魅力。进入大学，修读数学专业，能够强制被输入的人文知识更加难得，要求自律以坚持感性。因此，我希望读书与习作，输入与输出并肩同行，虽然不知道会得到什么结果，却也是令人期待的一段好景。

所以我的博客就诞生了，有一番自己掌控的小天地很是令人欣喜。我首先将2015年上半年在小本儿里记下的东西敲了进去，而后胡言乱语至今，延续至今后。进入大学，更希望把这片天地扩展开来，所以今后可能会将学业上的记录、我个人在其它学问上面的涉猎也记算在内，比如数学知识、计算机知识，再比如英语等等等等。

大学入学才四个月不到，走进深交的朋友还没有遇到，与高中班里的同学还联络甚频。谦虚盛行的班风让旧友之间的谈论都是对自己在新环境中对自我表现的不满，也未尝不是件好事。就让我们一起读书、讨论，再论明天。

技术上的辗转反侧

高三有了搭建博客的念头之后，便在偶得闲暇时候上网查找好的平台和搭建方法，了解到不少东西。直到毕业前夕，我所了解到的知识是，如果要搭建博客，大体的方向有几个选择：

  1. 门户网站的博客平台（比如网易博客、新浪博客、Lofter、QQ空间皆是此类）
  2. 提供独立网站空间的博客供应商（比如FarBox、Ghost、Wordpress.com）
  3. 自己购买域名和服务器空间，并上传一套程序来搭建博客。需要的程序也有这么两种选择：
    1. 利用动态网页和数据库构建网站的大型博客程序（比如WordPress）
    2. 静态网页生成器（比如Jekyll、Octopress、Hexo）

而我思量再三，决定摒弃第一条最简单的路径，因为这些门户网站提供的平台大多自由度低，不可以按自己的喜好随意定制。我的方式是曲线救国。我在6月20日左右先注册并搭建了一套 FarBox 博客，并通过鼓捣可乐橙提供的开放源代码的FarBox主题风格，逐渐钻研并少许熟络了诸如 Git、Github 和 HTML/CSS 等的前端开发知识。除此之外，当前大行其道的标记语言 Markdown 是创作独立博客必不可少的一个工具，它也极其简单，我捎带就学起来，用起来。

即便如此，FarBox 的使用方式及其简便，如果没有计算机基础，单靠网站提供的样式就能搭建出很精美的博客网站。我在这里真心打个广告，如果读者也有一个搭建独立博客的愿望，我推荐首先以 FarBox 作为开始和尝试，不愿折腾的可以沉下心来写文章，愿意折腾的，它也留下了很好的可定制性，让我们在实践中学习。

我的 FarBox 博客在6月底之前近乎完成，我开始了在这个小天地里写东西的假期生活。与此同时，我的折腾却也没闲下来，思量着有一天能拥有自己的服务器空间和独立的域名。于是7月初的时候我在阿里云网站上购买了首年免费的服务器空间和每年39元独立域名”www.mblquincy.cn”，也就是“梦不落Quincy”。于是我首先把自己的域名通过 DNSPod 解析到了我的 FarBox 博客。然后呢，我的服务器空间留下，给我的计算机知识学习做一个调试，也便于最终将我的博客迁移到真正的服务器空间上。因为阿里云的服务器空间和域名需要向工信部做备案才可以正常使用，几个月来一直懒于去备案点照个相，所以作罢。待我哪日有空去备案，最终的目的也就极快完成。

所以，这几个月来我也在为真正的独立空间做着种种准备。11月，通过在 Github 上与各路神仙取经，我认识了静态网页生成器 Hexo，这是一个台湾大学生开发出来的一套系统，需要一定的计算机知识才可以搭建完善。有了鼓捣 FarBox 的经验，我很快熟悉了这一套系统，并在 Github 提供的免费服务器空间 Github-Pages 上进行学习和调试。最终我使用了 Github 上好评无数的iissnan开发的 Hexo 主题 NeXT 完善了我的博客。

于是我的博客就变成了现在这副模样，目前 FarBox 博客已停止更新，Hexo 博客正在 Github-Pages 中暂时存放，而我的域名
“www.mblquincy.cn”也重新定向到了现在的博客上。年末之前我会去把阿里云的空间备案，然后将文件搬过去，整个博客在搭建技术上也就稳重下来了。

除此之外，为了在博客中实现特定的内容，我还涉猎了一些其它的琐碎知识。比如要在网页中显示数学公式内容而需要的工具 MathJax，它可以让网页文档中插入 $\mathrm{\LaTeX}$ 代码从而实现公式显示，等等诸如此类。

这就是我的博客搭建历程了，看起来学了不少东西，但也都是零零碎碎，左看看右看看也就熟悉不少。自己在这方面收获欣然，欣喜于能够有能力把自己对博客的想法实现出来。如果哪位伙伴也有愿望去搭建一番自己的博客，我也愿知无不言，提供帮助。

前路未完待续

博客运行半年有余，已成为我读书、思考的一个小空间，希望今后能够坚持做下去，记录自己。下面仍旧贴上开博建言，与自己共勉，与伙伴共勉。

不知道从什么时候开始，习惯了用起兴开始一篇文章，就像在敲这一段的时候，胸中似乎有一段难以言说的景致，非要我说出来，才能提起下面的话。可它现在像一块猩红色的淤血，被我堵着在胸口，我不让它跃出来。

也许我会永远感激这个决定，在这个午后，没有任何东西来搅动我。我把半年来，那个本子上的随写，都敲了上来。我决定宣告我这个博客的开始。博客，我从前是做过的。碍于学业，碍于精力，我一次次作罢。其实我很爱这一种方式。这个地方说公开却又私密，正是恰到好处的，一个寄托。

对，寄托。任何一个人都需要寄托，敏感的人更加需要。我从来会听到人们看过某个作者的心情随笔，便叫嚣着“矫情”之类的词语；可我们却不能否认有那样一些人，思考细致，情感丰富，没有寄托就会，溢出。也许我正是这样一种人。经年以来，一只一只的本子，或大或小，记载下我芜杂的心绪。

起初，我想，大概这种记录只不过是备受压力的生活中的一盏小小灯光，人们都会这样做。很多人写日记，看的人感动。我也是其中一个爱写“日记”的小朋友。我曾怀疑，这种我所珍惜的灵性，会不会在高考结束的那一刻消散殆尽，我再也不会“矫情”下去。但现在我知道，这种思索的习惯已经融在了我的骨血里，在能看得到的未来里，我也许会与这种思绪一起走下去。
我把这种思绪称作紫色的气，就是涂着口红巫婆用满是皱纹、指甲寸长的手随意挥出来的那那种。
